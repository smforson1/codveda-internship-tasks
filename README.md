# Codveda Internship Tasks

This repository contains the tasks for the Codveda Full-Stack Development Internship.

## Essential Commands

### Basic Terminal Commands

| Command | Description | Example |
|---------|-------------|---------|
| `ls` (Mac/Linux) / `dir` (Win) | List files and directories | `dir` |
| `cd <path>` | Change directory | `cd level1` |
| `mkdir <name>` | Create a new directory | `mkdir project` |
| `touch <file>` (Mac/Linux) / `type nul > <file>` (Win) | Create a new file | `type nul > app.js` |
| `rm <file>` (Mac/Linux) / `del <file>` (Win) | Remove a file | `del old_file.txt` |
| `cp <source> <dest>` (Mac/Linux) / `copy <source> <dest>` (Win) | Copy a file | `copy file.txt copy.txt` |
| `mv <source> <dest>` (Mac/Linux) / `move <source> <dest>` (Win) | Move or rename a file | `move old.txt new.txt` |

### Basic Git Commands

| Command | Description |
|---------|-------------|
| `git init` | Initialize a new Git repository |
| `git status` | Check the status of files |
| `git add <file>` | Stage a file for commit |
| `git add .` | Stage all changes |
| `git commit -m "message"` | Commit changes with a message |
| `git branch` | List branches |
| `git checkout -b <branch>` | Create and switch to a new branch |
| `git checkout <branch>` | Switch to an existing branch |
| `git push origin <branch>` | Push changes to remote repository |
| `git pull origin <branch>` | Pull changes from remote repository |
